# AIO - AI Agent Manager

## Install

**Termux (Android):** 

`pkg install git python && git clone https://github.com/seanpattencode/aio && cd aio && python aio.py install`

**Windows/Linux/Mac**

If windows:

Install WSL and a distro:
https://learn.microsoft.com/en-us/windows/wsl/install
note ubuntu is recommended.
`sudo apt update && sudo apt install git python3 -y`

then

`git clone https://github.com/seanpattencode/aio && cd aio && python3 aio.py install`

## Run
Type aio

## Warnings:
the aio command may not work properly if your directory has a . in the front of it.
aio push may say failure on mac but work.
