# Quick Start Guide

## claude1.py

```bash
# Add task
python3 claude1.py add-task "task_name" "command"

# Run worker
python3 claude1.py worker

# Check status
python3 claude1.py status
```

## claudeCode2.py

```bash
# Add task
python3 claudeCode2.py enqueue "task_name" "command"

# Run worker
python3 claudeCode2.py worker

# Check stats
python3 claudeCode2.py stats

# Test workflow
python3 claudeCode2.py test
```
