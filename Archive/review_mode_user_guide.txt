REVIEW MODE USER GUIDE
======================

COMMAND: aio review

PURPOSE: Systematically review all worktrees that are no longer actively running

WORKFLOW:
1. Run `aio review`
2. For each worktree:
   - Automatically opens in tmux session
   - Review the terminal output/logs
   - Press Ctrl+B D to detach
   - Use inspection commands to understand changes
   - Take action (push, delete, keep, or stop)

TYPICAL REVIEW PATTERNS:

Pattern 1: "Cleanup Commit"
- See in log: "cleanup files"
- Check diff: 2000+ deletions
- Decision: Press 2 (delete without push)

Pattern 2: "Unfinished Work"
- See in git status: ?? new_files
- Check with l: partial implementation
- Decision: Press 3 (keep for later)

Pattern 3: "Complete Feature"
- See in log: feature commits
- Check status: Clean
- Decision: Press 1 (push and delete)

Pattern 4: "Empty Worktree"
- No new commits
- No changes
- Decision: Press 2 (delete)
